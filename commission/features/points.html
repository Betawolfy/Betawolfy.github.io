Point System

A simple points system, with a leaderboard! Can be used for a lot of things!
Use a internal API.

Here's how The point system works.

This module comes with 4 main commands:

  • /points-add
    •
  • /points-remove
    •
  • /points
  • /leaderboard

1. /points-add

Syntax: /points-add user:{@user} numberPoint:{amount}
Permission required: permission.points.add or permission.all

This commands adds points to a user.
It can be used to reward a user for attending an event, or if the member filled a quest.

2. /points-remove

Syntax: /points-remove user:{@user} numberPoint:{amount}
Permission required: permission.points.remove or permission.all

This commands removes points to a user.

3. /points

Syntax: /points user:{@user}
Permission required: No permission required.

This commands shows the points of a user.
It can be used to see how many points a user have.
